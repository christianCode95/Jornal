# 📰 Jornal

This is a newspaper project, which has a simple newspaper format.

## 💡 How does it work?

- when you click on the "next page" button, it makes the animation of turning the page, as if it were turning a page of a newspaper;
- The animation is done by CSS, with rotation and perspective;
- The newspaper has 6 pages in total, with a cover and 5 more pages.
